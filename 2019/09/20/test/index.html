文件注意知识点

1. 文本文件

1.1 文本文件的打开与关闭

  • open()：文件分隔符要写成 “ \ “，例如：f = open("D:\\py\\a.txt", 'r')
  • open()中的模式注意点：
模式 功能 用法
r+ 为读写打开一个文本文件 一般先读后写，否则直接写原文件会被覆盖
w+ 为读写创建一个新的文本文件 一般先写后读，否则会所读内容为空
a+ 为读写打开一个文本文件 定位到文件末尾，原有内容不消失

1.2 文本文件的读写操作

1.2.1 read([count])
  • 读取当前位置到文件结尾的所有内容
  • 一般刚打开定位在文件开头，故读取所有内容
  • 读取的所有内容构成一个字符串返回
  • 若有参数，读取count个字符！！！
    1.2.2 readline() 和 readlines()
    • readline()：读取一行内容，读取的为字符串，一般带有换行符
      • 使用str[:-1]（换行符必存在）
      • 使用 line.rstrip()（换行符可以不存在）来去除换行符
    • readlines()：读取所有行，返回每一行（可能有换行符）内容构成的列表
1.2.3 write()
  • write不换行，需要换行要加换行符
1.2.4 writelines(lst)
  • 参数为列表，将列表中元素依次写入文件，换行符手动加

不存在writeline方法！！！

2. 二进制文件

2.1 文件的定位

  • tell()：返回文件当前的位置

  • seek(偏移[,参考点])：更改当前文件位置

    • 参考点：0（开始处），1（当前位置），2（末尾）
    • 文本文件只限制相对于文件起始位置进行移动seek(*,0)，否则只能seek(0,1)和seek(0,2)用于定位当前位置和文件末尾
    • seek(0,0)：返回文章开头

3. 文件管理方法

os模块

  • os.rename(“当前文件名”, “新文件名”)：文件重命名
  • os.remove(“文件名”)：删除文件
  • os.mkdir(“目录名”)：创建新目录
  • os.chdir(“目标目录”)：切换目录
  • os.getcwd()：显示当前目录
  • os.rmdir(“待删除目录名”)：删除的目录必须是空目录
