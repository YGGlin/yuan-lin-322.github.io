Leetcode Records: Palindrome Number

发表于 2019-09-24 分类于 leetcode

Problem

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数

Examples

输入 输出 说明
121 true
-121 false 从左向右读为 -121；从右向左读为 121- ；因此不是一个回文数
10 false 从右向左读, 为 01 ；因此它不是一个回文数

进阶：你能不将整数转为字符串来解决这个问题吗？

Solution

1、字符串方法：

  • 将整型转化为字符串：to_string(number)
  • 对字符串逆序：reverse(str.begin(),str.end())
  • 将逆序后的字符串转回整型：atoi(str.c_str())

16 ms 76.82%  8.2 MB 81.50%

class Solution {
public:
bool isPalindrome(int x) {
// 若x为负数或者x的最后一位为0但x本身不为0，肯定不是回文数
if(x < 0) return false;
int res = 0, tmp = x;
while(tmp != 0) {
// 防止溢出，如果发生溢出，那么当前x一定不是回文数
if (res > INT_MAX/10 || (res == INT_MAX / 10 && (tmp % 10) > 7)) return false;
res = res*10 + (tmp % 10);
tmp = tmp / 10;
}
if(res == x) return true;
else return false;
}
};

class Solution {
public:
bool isPalindrome(int x) {
if(x < 0 || x % 10 == 0 && x != 0) return false;
int rev = 0;
// 当rev不再小于x时，认为已经到达x的一半位数了
while(x > rev) {
rev = rev*10 + x % 10;
x /= 10;
}
// 考虑x的数位可能是奇数
return x == rev || x == rev / 10;
}
};

文件注意知识点

1. 文本文件

1.1 文本文件的打开与关闭

  • open()：文件分隔符要写成 “ \ “，例如：f = open("D:\\py\\a.txt", 'r')
  • open()中的模式注意点：
模式 功能 用法
r+ 为读写打开一个文本文件 一般先读后写，否则直接写原文件会被覆盖
w+ 为读写创建一个新的文本文件 一般先写后读，否则会所读内容为空
a+ 为读写打开一个文本文件 定位到文件末尾，原有内容不消失

1.2 文本文件的读写操作

1.2.1 read([count])
  • 读取当前位置到文件结尾的所有内容
  • 一般刚打开定位在文件开头，故读取所有内容
  • 读取的所有内容构成一个字符串返回
  • 若有参数，读取count个字符！！！
    1.2.2 readline() 和 readlines()
    • readline()：读取一行内容，读取的为字符串，一般带有换行符
      • 使用str[:-1]（换行符必存在）
      • 使用 line.rstrip()（换行符可以不存在）来去除换行符
    • readlines()：读取所有行，返回每一行（可能有换行符）内容构成的列表
1.2.3 write()
  • write不换行，需要换行要加换行符
1.2.4 writelines(lst)
  • 参数为列表，将列表中元素依次写入文件，换行符手动加

不存在writeline方法！！！

2. 二进制文件

2.1 文件的定位

  • tell()：返回文件当前的位置

  • seek(偏移[,参考点])：更改当前文件位置

    • 参考点：0（开始处），1（当前位置），2（末尾）
    • 文本文件只限制相对于文件起始位置进行移动seek(*,0)，否则只能seek(0,1)和seek(0,2)用于定位当前位置和文件末尾
    • seek(0,0)：返回文章开头

      2.2 文件的读写

      2.2.1 read([count]) 和 write()

  • 若read()有参数，读取count个字节！！！

  • chr()：返回ascii码表示的字符

  • ord()：返回字符对应的ascii码

  • 2.2.2 struct模块（处理二进制）

    import struct

  • pack()：把整形（或者浮点型）打包成二进制的字符串使其可以直接写入文件

  • unpack()：把二进制的字符串解包为数

import pickle

  • 直接将python对象写入文件，不需要将对象先转为字符串再写入
  • dump(数据, 文件对象)
  • load(文件对象)：

补充eval知识点
  • 将以字符串形式表现的python对象转化为原来的对象类型
  • 先导0会导致eval出错，要使用int：eval('003')× => int('003')
  • 把字符串的内容作为对应的python语句来执行

3. 文件管理方法

os模块

  • os.rename(“当前文件名”, “新文件名”)：文件重命名
  • os.remove(“文件名”)：删除文件
  • os.mkdir(“目录名”)：创建新目录
  • os.chdir(“目标目录”)：切换目录
  • os.getcwd()：显示当前目录
  • os.rmdir(“待删除目录名”)：删除的目录必须是空目录
